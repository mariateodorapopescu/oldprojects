Numărul de oxidare. Reguli la stabilirea numărului de oxidare

1) Numărul de oxidare al substanțelor aflate ȋn stare elementară este egal cu 0. 


0        0        0         0       0       0
Al       H2      Cl2        O2      I2     Cu
2) Numărul de oxidare al hidrogenului ȋn compușii cu nemetalele este +1, iar ȋn compușii cu metalele este -1. 

+1        +1        +1       -1      -1
H2O      H2SO4     NaOH     NaH     CaH2
3) Numărul de oxidare al oxigenului este -2 ȋn majoritatea compușilor, cu excepția peroxizilor unde este -1. 
-2           -2          -2        -1       -1
H2O         H2SO4       NaOH      NaO2     H2O2
4) Suma numerelor de oxidare ȋn substanțele compuse neutre este egală cu 0. 

+1 -2                       +1 -2 +1
H2  O                       Na O  H 

+1*2-2=0             +1-2+1=0
5) Numărul de oxidare al ionilor monoatomici este egal cu sarcina ionului. 

+1       +2            -1
Na+      Ca2+          Cl-
6) Numărul de oxidare al ionilor poliatomici este egal cu sarcina ionului. 
       
        x(+5) -1 
        N    O3-           
        x-2*3=-1 
        x=+5
