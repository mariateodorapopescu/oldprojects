Tipuri de catene de atomi de carbon

Catenele nesaturate conțin cel putin o legătură dublă sau triplă, iar cele saturate conțin numai legături simple.

Tipuri de atomi de carbon:
a) Carbon nular (nu se leagă de niciun alt atom de carbon) 

	
		   H
		   |
		 H—C—Cl              	CH3—O—CH3
		   |
	           H
b) Carbon primar (se leagă de un alt atom de carbon) 
	
		 | |
		—C—C—
	         | |
c) Carbon secundar (se leagă de alți doi atomi de carbon) 
	  
		 | | |
		—C—C—C—
	         | | |
d) Carbon terțiar (se leagă de alți trei atomi de carbon) 
	
		 | | |            
		—C—C—C—
		 | | |
		  —C—
		   |
e) Carbon cuaternar (se leagă de alți patru atomi de carbon) 
	
		 |
		—C—
		 | | | |
		—C—C—C—C—
		 | | | |
		—C—
