Legile gazelor

Parametrii care influențează starea gazoasă sunt: p, v, T.

T = t + 273, unde t este temperatura exprimată ȋn grade Celsius, iar T este temperatura exprimată ȋn Kelvin

(p+V) / T = ct

1 -> 2
p1 * V1 / T1 = p2 * V2 / T2 - legea generală a gazelor

Dacă presiunea rămȃne constantă:
p1 = p2 -> V1 / T1 = V2 / T2

Dacă volumul este constant:
p1 / T1 = p2 / T2

Dacă Temperatura este constantă:
p1 *V1 = p2 * V2

Un gaz se află ȋn condiții normale dacă:
p0 = 1atm
t0 = 0 grade celsius

Condiții standard
p = 1atm
T = 25 grade celsius

Volumul unui mol din orice gaz aflat ȋn condiții normale are valoarea de:
V0 = Vm = 22,4l

n = V / Vm ,pentru condiții normale

p0 * V0 / T0 = P * V / T ,pentru 1 mol de gaz

R = constanta generală a gazelor
R = 0,082 atm * dm3 / kg * mol

P = mmHg
R = 62400 mmHg * cm3 / k * mol

P * V = R * T pentru 1 mol de gaz
P * V = n * R * T pentru n moli => ecuația de stare a gazelor(se aplică ȋn general, cȃnd gazul nu se află ȋn condiții normale)

Relația de calcul a densității unui gaz sau a unui amestec de gaze care nu se află ȋn condiții normale: S = p * M / R * T
Dacă gazul care nu se află ȋn condiții normale, se folosește formula: S = M / Vm ,pentru 1 mol de gaz.
Unde
M - masa moleculară
Vm - V molar
S0 = densitatea absolută (densitatea ȋn condiții normale)

Legea lui Dalton
Presiunea unui amestec de gaze este egală cu suma presiunilor parțiale ale gazelor componente.

P = p1 + p2 + … + pn
Pi = xi * P
Xi = fracția molară a componentei respective
xi = ni / ntotal
pi = ai / 100 * P
ai = procentul volumetric din componenta respectivă
La gaze, procentele de volum sunt egale cu procentele de moli.

Legea lui Avogadro
Volume egale de gaze diferite aflate ȋn aceleași condiții de presiune și temperatură, au același număr de molecule.
Numărul lui Avogadro=6,022*1023 particule/mol
1 mol din orice substanță conține numărul lui Avogadro atomi, ioni sau molecule.

Masa moleculară medie a unui amestec gazos se poate calcula cu următoarele relații:
1) M = x1 * M1 + x2 * M2 + … + xn * Mn
2) M = a1 / 100 * M1 + a2 / 100 * M2 + … + an * Mn
a1, a2, a3, …, an= procentele de volum ale componentelor
